# PRPO Project Page

This repository hosts the project page for the PRPO paper (ICLR 2026).

## Local preview

Open `index.html` directly, or run a local static server:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080.

## Deployment

This site is intended for GitHub Pages on the custom domain `prpo.github.io`.
